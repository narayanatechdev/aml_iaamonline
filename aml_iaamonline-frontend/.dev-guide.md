# Development Quick Reference

## Start Development

```bash
cd /Users/narayana/Herd/aml_iaamonline-frontend
npm install     # First time only
npm run dev     # Start development server on http://localhost:3000
```

## Project Stack

- Framework: Next.js 16 (React 19, TypeScript)
- Styling: Tailwind CSS v4
- UI Components: shadcn/ui
- Theme: Luma (Light), Indigo (Primary), Taupe (Secondary)

## File Structure

```
app/
├── globals.css          - CSS variables and theme
├── layout.tsx           - Root layout
└── page.tsx             - Home page

components/
├── ui/                  - Reusable UI components
│   ├── button.tsx
│   ├── card.tsx
│   └── input.tsx
└── layout/              - Layout components

lib/
├── api-client.ts        - API utilities
└── cn.ts                - Class name utility
```

## Call Backend API

```typescript
import { apiClient } from '@/lib/api-client';

// GET
const data = await apiClient.get('/users');

// POST
const response = await apiClient.post('/users', { name: 'John' });

// PUT
await apiClient.put('/users/1', { name: 'Jane' });

// DELETE
await apiClient.delete('/users/1');
```

## Use UI Components

```tsx
import { Button } from '@/components/ui/button';
import { Card, CardHeader, CardTitle, CardContent } from '@/components/ui/card';
import { Input } from '@/components/ui/input';
```

## Commands

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm start        # Run production build
npm run lint     # Check code style
```

## Environment Variables

Edit `.env.local` to configure:

```
NEXT_PUBLIC_API_URL=http://localhost:8000/api
NEXT_PUBLIC_APP_NAME=AML IAA Online
```

Ensure Laravel server is running at the API URL.
